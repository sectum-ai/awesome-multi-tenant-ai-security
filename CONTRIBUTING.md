# Contributing

Thanks for helping curate this list. Please make sure your contribution fits
the scope: resources on **multi-tenant / cross-tenant AI security** — keeping
one tenant's or one user's data from reaching another across the AI surface.

Guidelines:

- One item per line: `- [Name](https://link) — short, factual description.`
- Keep descriptions neutral and free of marketing language.
- Put the item in the most relevant section (add a section if none fits).
- Verify the link works; alphabetize within a section where practical.
- Prefer primary sources (papers, advisories, specs, source repos).

Open a pull request with your addition.
